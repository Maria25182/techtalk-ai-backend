# TechTalk AI - Frontend

Frontend de TechTalk AI con Web Speech API.

## Setup

1. Editar `index.html` línea 270
2. Cambiar `API_URL` a tu URL de Railway:
   ```javascript
   const API_URL = 'https://tu-backend.up.railway.app';
   ```

## Deploy en Vercel

1. Subir a GitHub
2. Conectar repo en Vercel
3. Deploy automático

## Uso Local

Abrir `index.html` en Chrome (con backend corriendo en localhost:8000)
